# Tipos de dados

Conforme o ECMAScript standards temos 9 tipod de dados:

*Data types
    * Primitive / Primitive value
    * Structural
    * Structural Primitive

## Primitivos 

* String 
* Number 
* Boolean 
* undefined
* Symbol
* BigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function
